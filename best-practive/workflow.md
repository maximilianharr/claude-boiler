2. ``/grill-me ${PROMPT}``

4. `shift tab` to auto mode
5. ```Review this project for completeness, accuracy, improvement, security etc. Run that review by Sonnet4.6 and get its input into your findings to see if you're missing anything. Display your findings in a list ordered by priority. Fix every item in the list. then repeat this process until both you and Sonnet4.6 agree that the only remaining items have diminishing return.```
